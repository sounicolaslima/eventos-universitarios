# 🎓 Plataforma de Gestão de Eventos Universitários

Sistema web para gerenciamento de eventos acadêmicos, permitindo criação, inscrição e acompanhamento de participação.

## 🚀 Como rodar o projeto

**Clonar repositório:** 
```bash
git clone <link>
````

**Backend**
```bash
cd backend
npm install
node src/app.js
````

**Frontend**
```bash
cd ../frontend
npm install
npm run dev
````
